# Entrega Parcial 2 | Tarea 1

Para esta tarea, se le pide que clonar el repositorio de Github y comenzar a implementar los distintos tipos de *Carta*:
de unidad y de clima. Para esto, refiérase a la sección 2.3 del documento que plantea el proyecto semestral, a excepción
de las funcionalidades que involucren al *Tablero* y/o las *Clasificaciones* de las cartas.

En específico, se le pide:

1. Crear una rama llamada ``tarea-1/entrega-parcial-2``.
    Recuerde hacer *checkout* a esta rama para realizar la tarea.
2. Identificar qué elementos de los mencionados en el enunciado deben ser clases concretas o abstractas y cuál es la
    interfaz/trait (o en su defecto clase abstracta) que exponen.
3. Definir los casos de prueba para cada método y variable que considere necesario.
    Para esto tenga en cuenta cuáles son los casos de borde y los casos normales (se sugiere que
    tenga **al menos** un caso de prueba para cada caso normal y para caso de borde).
4. Implementar los tests a partir de sus casos de prueba.


**IMPORTANTE**: Para trabajar en su tarea, debe abrirla en IntelliJ desde la carpeta que clonó, es decir, la que
tenga el nombre *tarea-0-YourGithubName*. Hacerlo desde la carpeta que contiene a esta no permitirá el correcto
funcionamiento de la aplicación.

Por *U-Cursos* debe entregar un único archivo llamado ``entrega-parcial-2.txt`` con el siguiente contenido:

```
Nombre: <nombre completo>
Pull Request: <link al pull request>
```

El link correcto de un pull request es de la forma:
```
https://github.com/dcc-cc3002/tarea-0-YourGithubName/pull/x
```
Con *YourGithubName* siendo su nombre de Github, y *x* un número que indica el ID de su pull request.

## Evaluación

Para la evaluación de esta tarea se considerará el avance que hayan logrado en la implementación de
los tipos de carta y sus tests, sin importar si estos presentan errores.

La realización de esta tarea es **obligatoria** y corresponde a 0.5 puntos de la nota final de la Tarea 1.
