# desarrollo_web_felipe_barrientos
Se realizo uso de funciones creadas en auxiliar, creando getters en un principio para luego validar con funciones, para luego validar todo en una funcion que englobe la funciones anteriores.
Se prefirio escribir manualmente para evitar desincronizaciones entre datos a costa de perder eficiencia, siendo mas engorroso, pero a su vez mas fácil de editar.
Muchas de las funciones de validación se buscaron en internet y se implementaron posiblemente no de la manera mas eficiente
